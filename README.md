# Spang 👩‍👦‍👦

## 선정 이유 ❓
계속 되는 코로나시대에도 같이 운동을해 극복해 가자는 생각으로<br />
체육관을 빌려 운동을 같이하면 어떨까 라는 생각으로 이 주제를 정하게 되었습니다.<br />
